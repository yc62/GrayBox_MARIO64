# Graybox Mario 64

A classic 3D platforming experience distilled into a minimalist "graybox" style, running directly in your web browser. Built using [Three.js](https://threejs.org/), this project features custom physics, enemy AI, and interactive gameplay mechanics. 

## 🎮 Gameplay Features

* **Platforming Action:** Control the red cube (the player) across five floating green islands suspended over a blue ocean. Don't fall, or it's game over!
* **Dynamic Enemies:**
    * **Goombas (Brown):** Slower-moving enemies that patrol the islands. 
    * **Koopa Troopas (Green):** Faster-moving enemies.
    * **Piranha Plants:** Stationary enemies that bob up and down, shooting red fireballs if you get too close. 
    * *Tip:* You can defeat Goombas and Koopa Troopas by jumping on top of them!
* **Collectibles & Score:** * Collect spinning yellow boxes for **+1 coin**.
    * Stomp enemies for **+3 coins**.
    * Reach the final yellow star to win the game and earn a **+10 coin** bonus!
* **Custom Physics:** Features tuned gravity, terminal falling velocity, and smooth camera tracking that follows the player's movements.

## ⌨️ Controls

* **Move:** `W`, `A`, `S`, `D` or `Arrow Keys`
* **Jump:** `Spacebar`
